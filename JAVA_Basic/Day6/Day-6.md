# Day 6: Pattern Printing Programs

This folder contains Java programs that demonstrate various pattern printing techniques using nested loops. These are part of a Java basics tutorial series.

## Files

- **ButterflyPattern.java**: Prints a butterfly pattern using asterisks.
- **DiamondPyramid.java**: Prints a diamond-shaped pyramid pattern.
- **NumberPyramid.java**: Prints a pyramid pattern with numbers.
- **PalindromicPattern.java**: Prints a palindromic pattern.
- **SolidRombus.java**: Prints a solid rhombus pattern.

## How to Run

These programs are in the `JAVA_Basic.Day6` package. To compile and run:

1. Navigate to the project root directory (`c:\Users\ASUS\Desktop\Java`).
2. Compile: `javac JAVA_Basic/Day6/*.java`
3. Run: `java JAVA_Basic.Day6.ClassName` (replace ClassName with the specific class, e.g., `ButterflyPattern`)

## Requirements

- Java Development Kit (JDK) 17 or higher (based on the terminal output).
- The programs use standard Java libraries and do not require additional dependencies.
